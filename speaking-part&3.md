  - Shanghai is a **metropolis**.

### Basic Answer Structure

- The **TS+SD** Principle
  - **T**opic **S**entence
    - to give a clear, simple, ***direct*** answer
  - **S**upporting **D**etails
    - to elaborate and make your meaning ***clearer***
    - to ***communicate effectively***
    - to show your linguistic ability

### Topic Sentence

- no need to repeat the question （没有必要重复问题）

- no need to paraphrase the question (没有必要同义替换)

- **MUST** answer accordingly(根据题目类型回答)

  | Do you like to swim?                | No, I don't.                  |
  | ----------------------------------- | ----------------------------- |
  | Where is your hometown?             | Dengfeng./come from Dengfeng. |
  | Which do you prefer, tea or coffee? | I like tea better.            |
  | Would you continue to live there?   | No, I don't think so.         |

> Linking words must be correct!

| 加相似或等同观点           | 加对立观点             | 解释或重述观点      | 强调观点 |
| -------------------------- | ---------------------- | ------------------- | -------- |
| and / also                 | but / yet              | in other words, ... | indeed   |
| besides (除了应该用except) | on the other hand...   | in particular, ...  | in fact  |

| another...                 | in spite of ...        |                     |          |

### Basic Requirements

> Fluency & Pronunciation

- Control length of answer
  - **20-30 sec** for each question
  - about **3-5 sentences** (包括连接词connection words)
  - some answers should be long (e.g. reasons, preferences, comparison, experiences, descriptions, importance)
  - others don't need to do
  - be natural, don't talk ***too*** much
